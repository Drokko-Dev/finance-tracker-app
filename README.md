# Proyecto Finanzas - Monorepo

## Cómo empezar

1. Clona el repo.
2. Crea un archivo `.env` en `/backend` con tu `DATABASE_URL` de Supabase.
3. Ejecuta `docker-compose up --build`.
4. Frontend: http://localhost:5173 | Backend Docs: http://localhost:8000/docs

## Personal Finance Tracker

A full-stack application to monitor savings, expenses, and income. Built with FastAPI, React, and PostgreSQL using a clean architecture and Docker.
